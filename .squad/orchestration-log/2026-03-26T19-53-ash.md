## Recommendations

### 🚫 Remove from Fast-Path

1. Any fuzzy entity resolution — require 100% string match or bail to orchestrator
2. Color commands — route to orchestrator + skill with fixed prompt
3. Fan and HVAC modes — add patterns and orchestrator coordination
4. Temporal commands — timer-agent needed
5. Multi-room/group commands — orchestrator multi-skill coordination
6. Relative adjustments without value — needs relative brightness pattern

### ✅ Keep in Fast-Path

1. Exact-match on/off with known entity/area
2. Exact temperature set
3. Simple scene activation

### 🔧 Architectural Fixes

1. Add STT quality gate (coherence detection)
2. Entity name preservation rule in orchestrator (no translation)
3. Multi-turn task rehydration (durable conversation state)
4. Tighten fast-path confidence to reflect resolution quality
5. Entity resolution quality signal (abort on low-quality match)
